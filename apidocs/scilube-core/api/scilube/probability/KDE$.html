object KDE

Kernel Density Estimator for one-dimensional data. This is a port of kde.m at http://mathworks.us/matlabcentral/fileexchange/authors/27236. The output of this matches kde.m exactly. Help from kde.m:

"Gaussian kernel is assumed and the bandwidth is chosen automatically; Unlike many other implementations, this one is immune to problems caused by multimodal densities with widely separated modes (see example). The estimation does not deteriorate for multimodal densities, because we never assume a parametric model for the data."

Reference: Z. I. Botev, J. F. Grotowski, and D. P. Kroese (2010) Annals of Statistics, Volume 38, Number 5, pages 2916-2957.
