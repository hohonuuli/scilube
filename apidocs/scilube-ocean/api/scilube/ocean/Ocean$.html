  33.  def oc2v2(lu490: Double, lu555: Double): Double

    OC2V2 Modified ocean chlorphyll 2 algorithm

    OC2V2 Modified ocean chlorphyll 2 algorithm

    OC2V2 calculates ocean chlorphyll from in water upwelled radiance (Lu). The algorithm is from Stephane Maritorena <stephane@calval.gfsc.nasa.gov>

    It's modified in that the original algotithm uses: R = log(Rrs_490/Rrs_555) where Rrs is the remote reflectance ratio

    Be aware: R = log(Lu_490/Lu_555) as an approximation gives values that are closer to morel's algorythmn (at least for equatorial data).

    Note: the units of LU490 and LU_555 are not important. however they must be the same for both inputs

    lu490

    Upwelled radiance at 490 nm

    lu555

    Upwelled radiance at 555 nm

    returns

    chlorophyll (ug/L)
